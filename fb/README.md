# Telco - Next Best Offer

## Initiate this Project

- Installed **Node.js & npm**.
- Clone this Repo, using your terminal. `git clone https://github.com/MRHP1/Capstone-Asah`
- Navigate to the Project Directory. `cd fb/telco-frontend`
- Install Dependencies `npm install`
- Run Project `npm run dev`

## This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
